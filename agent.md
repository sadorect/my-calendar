# Personal Productivity Calendar Application - Progress Tracker

## Project Overview

Building a comprehensive Vue.js 3 calendar app with advanced features for personal productivity management.

## Technology Stack

- Vue.js 3 (Composition API)
- Vite
- Pinia
- Tailwind CSS
- FullCalendar Vue
- date-fns
- localforage
- And more...
